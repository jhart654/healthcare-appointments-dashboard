# healthcare-appointments-dashboard

## This project analyzes patient appointment data using Power BI to identify trends and optimize healthcare operations.

## Tools Used
* Power BI
* Data Visualization
* Data Aggregation

## Key Insights
* Tracked total appointment volume (200+ visits)
* Identified trends in patient appointments over time
* Evaluated provider utilization across doctors

## Skills Demonstrated
* KPI creation (Count aggregation)
* Time-series analysis
* Dashboard design and layout
